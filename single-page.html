Deploying a website by using Heroku

Heroku is a hosting platform where you can deploy dynamic applications in Rails, PHP, Node.js and Python web applications.
Prerequisites： In order to deploy a site you need a couple of things:1.Have git installed 2.Heroku Account 3.Download the Heroku Toolbelt – a command line application for managing your Heroku account 4.Run heroku login in your terminal or command prompt and fill in your Heroku credentials

Deploying your site: First, you need to navigate to your project in the command prompt. Then you can deploy your website as this order: 1. git init 2. git status 3. git add . 4. git commit -m " " 5. heroku login 6. heroku create 7. (add start script in the package) 8. git add package.json 9. git commit -m " add start script" 10. operate the mongodbLab on the website (convert the local DB to the remote DB) 10.git push heroku master
